## 🎯 Pro Tips

### Multi-Cursor Editing
1. Hold `Ctrl+Alt` and use `Up/Down` arrows to add cursors
2. Use `Ctrl+D` to select next occurrence of current word
3. Use `Ctrl+Shift+L` to select all occurrences at once
4. Press `Shift+Alt+I` to add cursors at end of each selected line

### Quick Navigation
1. `Ctrl+P` opens Quick Open - start typing filename
2. `Ctrl+Shift+P` opens Command Palette - access all commands
3. `Ctrl+Tab` cycles through open editors
4. `Alt+Left/Right` navigates through edit history

### Line Manipulation
1. `Alt+Up/Down` moves current line up or down
2. `Shift+Alt+Up/Down` duplicates line above or below
3. `Ctrl+Shift+K` deletes current line
4. `Ctrl+Enter` inserts line below without breaking current line

### Search & Replace
1. `Ctrl+F` for find in current file
2. `Ctrl+H` for replace in current file
3. `Ctrl+Shift+F` for find across all files
4. `Ctrl+Shift+H` for replace across all files
5. Use regex mode for advanced patterns

### Code Formatting
1. `Shift+Alt+F` formats entire document
2. `Ctrl+K Ctrl+F` formats only selected code
3. `Ctrl+/` toggles line comment
4. `Shift+Alt+A` toggles block comment
